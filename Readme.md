  <a id="console"></a>
<details>
<summary>Windows console compilation</summary>
<ul>
<li>Go to the project folder:</li>

    >cd ..\SearchEngine
    
<li>create a build directory:</li>

    >mkdir build
      
<li>Then go to the build directory and run CMake:</li>

    >cd build
    >cmake ..
    
<li>Then compile the project:</li>

    >cmake build ..

</ul>
